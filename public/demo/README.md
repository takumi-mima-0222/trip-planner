# Demo Files

このディレクトリにはデモ用のファイルを配置してください。

## 必要なファイル

### 動画ファイル（推奨）
- `tabikumi-demo.mp4` - デモ動画（16:9、30秒程度）
- `tabikumi-demo-poster.png` - 動画のポスター画像（サムネイル）

### 推奨仕様
- **解像度**: 1920x1080 または 1280x720
- **アスペクト比**: 16:9
- **動画長**: 20〜40秒
- **ファイルサイズ**: 10MB以下を推奨（Web最適化）

### 動画の内容
1. 入力画面（目的地・日程・スポットを入力）
2. 生成中の画面（AIが処理中）
3. 生成結果画面（日別タイムライン表示）
